## 什么是 Overlay？

**Overlay（套刻精度）** 指当前光刻层与前一层图案之间的对准精度。在芯片制造中，一个器件通常需要 20-60 层光刻，每一层都必须精确对准前面的层。

- **层间对准误差** = 当前层图案中心 vs 参考层图案中心的位置偏差
- **规格要求**：通常为最小 CD 的 1/3 ~ 1/5
- **3nm 节点**：Overlay 要求 < 2nm

## Overlay 测量标记

### Box-in-Box（盒中盒）

最常见的 Overlay 测量标记：

- 外层方框：参考层（前一层）图案
- 内层方框：当前层图案
- 测量内外框中心的位置偏差 → X/Y Overlay 误差

### AIM（Advanced Imaging Metrology）标记

- 光栅结构，衍射效率更高
- 适用于更小的 CD 节点
- 可同时测量多个 Overlay 分量

## Overlay 误差来源

| 误差类型 | 说明 |
|---------|------|
| **平移 (Translation)** | X/Y 方向的整体偏移 |
| **旋转 (Rotation)** | 层间相对旋转角度 |
| **缩放 (Scaling)** | 层间放大倍数差异 |
| **正交性 (Orthogonality)** | X/Y 轴不垂直导致的误差 |
| **高阶畸变** | 透镜畸变等非线性误差 |

## 实际影响

Overlay 误差的直接后果：

- **接触孔未对准** → 开路或高电阻
- **栅极偏移** → 沟道长度变化，性能偏移
- **金属层错位** → 短路风险

在明场/暗场检测设备中，Overlay 测量是**核心功能**之一。
