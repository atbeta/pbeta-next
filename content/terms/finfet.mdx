## 为什么需要 FinFET？

随着制程缩小到 22nm 以下，传统平面 MOSFET 面临严重的**短沟道效应**：

- 漏极电场穿透到沟道区 → 栅极失去对沟道的控制
- 亚阈值漏电流急剧增大 → 静态功耗失控
- 阈值电压随沟道长度漂移 → 器件一致性变差

**FinFET 的解决方案**：将沟道从平面立起来，让栅极从三面包围沟道（Tri-Gate），大大增强了栅极的静电控制能力。

## FinFET 结构

FinFET 的关键特征：

| 组件 | 功能 |
|------|------|
| **鳍 (Fin)** | 垂直的硅沟道，电流沿鳍的高度方向流动 |
| **栅极 (Gate)** | 包裹在鳍的三面，控制沟道的导通/关断 |
| **栅氧 (Gate Oxide)** | 栅极与鳍之间的绝缘层（High-k 材料） |
| **源极 (Source)** | 载流子注入端 |
| **漏极 (Drain)** | 载流子收集端 |
| **STI (浅槽隔离)** | 鳍之间的绝缘填充 |

### 关键参数

- **鳍宽 (Fin Width)**：鳍的横向宽度，影响短沟道效应控制
- **鳍高 (Fin Height)**：鳍的垂直高度，决定有效沟道宽度
- **栅极长度 (Gate Length)**：沿鳍方向的栅极长度，即有效沟道长度
- **鳍间距 (Fin Pitch)**：相邻鳍之间的距离

## FinFET vs 平面 MOSFET

| 特性 | 平面 MOSFET | FinFET |
|------|-----------|--------|
| 栅极控制 | 单面（顶面）| 三面（Tri-Gate）|
| 漏电流 | 高（短沟道效应）| 低 |
| 驱动电流 | 受限于沟道宽度 | 可通过增加鳍数提升 |
| 制程节点 | ≥ 28nm | ≤ 22nm |

## 从 FinFET 到 GAA

随着制程继续缩小到 3nm 以下，FinFET 的鳍宽已接近物理极限。下一代技术是 **GAA（Gate-All-Around）**：

- 沟道从鳍变为水平的纳米片（Nanosheet）
- 栅极从**四面**完全包围沟道
- 静电控制能力达到理论最优

多层纳米片的 3D 结构建模需要精确的几何计算，是工艺仿真工具（如 SEMulator3D、pythonOCC 等）的典型应用场景。
