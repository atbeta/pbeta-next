PicFast 是可一键部署的开源图床，提供易用的管理后台和开放 API，轻松对接上传工具、自动化工作流及各类 AI 应用。已发布到 Docker Hub 并提供完整文档站点 [picfast.dev](https://picfast.dev)。

## 技术栈

| 层级 | 技术 |
|------|------|
| 后端 | Go 1.26, Chi Router, pgx/v5, sqlc, JWT |
| 前端 | React 19, TypeScript, Vite, React Router, Tailwind CSS v4 |
| 数据库 | PostgreSQL 16 |
| 存储 | 本地 / S3 兼容 / 七牛 Kodo / 阿里云 OSS / 腾讯云 COS / WebDAV |
| 可观测 | Prometheus、健康检查 |

## 核心功能

- **多用户与游客上传** — 游客限额可配置，支持邮箱注册/验证
- **多存储策略** — 支持 6 种存储后端，按用户/相册灵活配置
- **管理后台** — 主题定制、用户管理、存储配置、维护工具
- **MCP 集成** — `@picfast/mcp` 包，AI Agent 可直接上传/管理图片
- **Docker 一键部署** — `xbeta/picfast` 镜像，Compose 模板开箱即用
- **ShareX / API Token 集成** — 支持常用截图上传工具
- **备份与恢复 CLI** — 内置 doctor/backup/inspect/restore/repair 命令

当前版本 v0.8.0，GPL-3.0 开源。
