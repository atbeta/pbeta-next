这是当前正在访问的网站。项目从零重建，目标是把个人实践、项目沉淀与长期内容创作整合到同一个可持续维护的平台中。

## 技术栈

- **前端**：Next.js 16 (Turbopack) + React 19 + Tailwind CSS v4
- **后端**：NestJS 11 + Prisma 7 + PostgreSQL 17
- **内容**：@content-collections + MDX + rehype-pretty-code
- **基础设施**：Railway（Web + API + PostgreSQL + Redis）

## 设计目标

以「数字花园」为核心表达方式，持续迭代已有内容，而不是一次性发布后沉底。

服务监控模块用于维护在线服务可观测性，后端 API 同时为后续工具化功能预留扩展空间。
