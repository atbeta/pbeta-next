`RelayCore` 是一个纯 Rust 编写的代理内核项目，目标是为 `RelayCraft` 提供更可控、更高性能的底层数据处理能力。当前该项目尚未公开到 GitHub，仍在内部持续迭代。

## 架构优势

- **性能优先**：以 Rust 异步 I/O 为基础，面向高并发代理场景优化。
- **安全与稳定**：利用 Rust 的类型系统与所有权模型降低运行时风险。
- **模块化设计**：按协议处理、过滤器与流量管线拆分，便于后续扩展。
- **面向 RelayCraft 集成**：作为长期演进方向，为上层产品沉淀核心能力。

当前页面用于记录项目定位与阶段进展，公开仓库与独立发布计划将在后续版本中补充。
