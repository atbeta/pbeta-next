RelayCore 是一个高性能 Rust 流量拦截引擎，定位为 mitmproxy 的现代替代方案。目前已发布 v0.1.2，托管在 [relaycraft/relay-core](https://github.com/relaycraft/relay-core)。

## 核心功能

- **HTTP/HTTPS MITM 代理** — 动态 TLS 证书生成
- **WebSocket 拦截** — 消息级检查、修改与重放
- **规则引擎** — 匹配 + 动作流水线（headers、body、status、redirect、mock、ratelimit）
- **Deno 脚本运行时** — 动态修改请求/响应/WebSocket
- **拦截断点** — 暂停实时流量、检查、修改后放行或丢弃
- **透明代理** — Linux TPROXY (TCP+UDP)，macOS PF (TCP)
- **流量脱敏** — 可配置的敏感字段遮蔽
- **审计追踪** — 控制面变更记录与持久化
- **Prometheus 指标** — 结构化指标快照

## 架构

```
Adapters (CLI, HTTP API, MCP, Tauri)
        │
relay-core-runtime (state hub, lifecycle, event bus)
        │
relay-core-lib (capture, proxy, MITM, TLS, rule engine)
        │
relay-core-api / relay-core-script / relay-core-storage
```
